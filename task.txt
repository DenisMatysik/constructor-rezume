Базовое тестовое задание на должность JS разработчика
Задача:
Создание генератора резюме на одной странице
Условия: Использовать чистый JS, не использовать различные плагины, библиотеки
Техническое задание:
1) Загрузка фото на страницу с визуальным отображением после загрузки
2) Создать 4-5 поля для ввода информации: ФИО, год рождения, телефон, почта. Все 
поля должны иметь соответствующую проверку на валидность
3) Создание блоков с информацией. Каждый блок должен иметь заголовок, в каждом 
блоке можно создавать неограниченное количество полей с заголовками.
Например:
Хобби <заголовок блока> - любимый спорт <заголовок поля>: бокс <содержание поля> - любимая книга<заголовок поля>: “Война и мир” <содержание поля>
